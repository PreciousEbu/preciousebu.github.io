Featured Projects

These end-to-end projects showcase my ability to move from raw data to actionable business insights, focusing on measurable results, technical rigor, and production readiness.

TELCO CUSTOMER CHURN PREDICTION

Customer retention is often more cost-effective than acquiring new customers, making early identification of at-risk customers a key business priority. In this project, I developed and evaluated machine learning models using demographic and service usage data to predict customer churn. After performing exploratory analysis, feature engineering, and model evaluation, the best-performing model achieved a test AUC of 0.847, providing reliable predictions to support targeted retention strategies. Built using R, the project demonstrates practical applications of predictive analytics for customer retention and business decision-making.

WALMART STORE SALES PREDICTION

Accurate sales forecasting enables retailers to optimize inventory, staffing, and promotional planning. In this project, I developed an end-to-end forecasting pipeline using historical Walmart sales, economic indicators, store metadata, and holiday effects. Multiple forecasting models—including SARIMAX, Prophet, and XGBoost—were evaluated using time-based validation and standard forecasting metrics (RMSE, MAE, and MAPE). Interactive Plotly dashboards were created to visualize sales trends, holiday impacts, and forecast performance, demonstrating the application of time-series analytics to business decision-making.

FINANCIAL RISK & FRAUD DETECTION SYSTEM

Financial institutions process thousands of transactions daily, making accurate fraud detection essential for minimizing losses while reducing false alarms. In this project, I built an end-to-end fraud detection pipeline that addressed severe class imbalance using SMOTEENN and implemented leakage-safe model validation. The final model achieved a ROC-AUC of 0.966 and an F1-score of 0.799, demonstrating strong predictive performance. Built in Python, this project showcases practical machine learning techniques for financial risk management.

CARDIOVASCULAR DISEASE CLASSIFICATION & RISK PREDICTION

Early identification of individuals at risk of heart disease can support timely intervention and improve health outcomes. This project applies machine learning techniques to predict heart disease risk using clinical data, incorporating feature engineering, class balancing, and cross-validation to build interpretable predictive models. The final models achieved 75% prediction accuracy while identifying key factors associated with cardiovascular risk. Developed in R, the project demonstrates the application of data science to healthcare decision support.

CUSTOMER SEGMENTATION DASHBOARD

Understanding customer purchasing behavior enables businesses to improve retention, increase customer lifetime value, and optimize marketing efforts. This project uses RFM (Recency, Frequency, Monetary) analysis to segment customers into actionable groups based on purchasing patterns. An interactive dashboard built with Python, Plotly, and Dash allows users to explore customer segments and engagement trends, supporting targeted marketing campaigns and revenue optimization.

GLOBAL SUICIDE RATE ANALYSIS

Suicide rates vary across countries, demographic groups, and time periods, making data-driven analysis essential for effective public health interventions. In this project, I explored global suicide rate data to identify demographic and temporal patterns through exploratory data analysis and visualization. The analysis highlights population groups at greater risk and demonstrates how data analytics can support evidence-based public health planning using Python.

SUPERSTORE SALES DASHBOARD

Business leaders rely on timely sales insights to monitor performance and make informed commercial decisions. This project presents an interactive Tableau dashboard that analyzes sales, profit, customer segments, and regional performance to uncover trends and opportunities. The dashboard enables users to monitor key performance indicators and supports data-driven decision-making through intuitive visual analytics.
